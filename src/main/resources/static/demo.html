Z-RAG 智能问答系统

基于 LangChain4j 0.29.1 的完整 RAG 解决方案

进入系统 查看状态

🚀 核心功能

  • 智能问答 - 基于文档的智能问答系统
  • 文档管理 - 支持多种格式文档的上传和管理
  • 知识管理 - 向量化存储和检索
  • 多模型支持 - 千问、OpenAI、Ollama
  • 重排优化 - 智能重排提升检索质量

🛠️ 技术栈

  • 后端 - Java 8 + Spring Boot 2.7.18
  • AI框架 - LangChain4j 0.29.1
  • 存储 - MinIO 对象存储
  • 前端 - HTML5 + CSS3 + JavaScript
  • 构建工具 - Maven

📡 API 演示

1. 检查系统状态

GET /api/rag/status

2. 上传文档

POST /api/rag/upload Content-Type: multipart/form-data file: [文档文件]

3. 智能问答

POST /api/rag/query Content-Type: application/json { "query": "你的问题" }

4. 文档检索

GET /api/rag/retrieve?query=检索关键词

📚 使用说明

  1. 启动系统：./start-qwen.sh 或 ./start-ollama.sh
  2. 访问系统：http://localhost:8080
  3. 上传文档：在文档管理页面上传您的文档
  4. 开始问答：在智能问答页面与系统对话
  5. 管理知识：在知识管理页面查看向量数据

🔧 配置说明

在 src/main/resources/application.yml 中配置：

# 千问模型配置（推荐） models: qwen: api: key: your-qwen-api-key base: url: https://dashscope.aliyuncs.com/api/v1 model: qwen-turbo default: provider: qwen
